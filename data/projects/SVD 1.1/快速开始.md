# SVD 1.1 快速开始指南

## 5 分钟上手 SVD 视频生成

### 步骤 1: 部署服务 (2 分钟)

```bash
cd "data/projects/SVD 1.1"
modal deploy svd_deploy.py
```

等待模型下载和部署完成。

### 步骤 2: 访问 Web 界面 (1 分钟)

部署成功后，终端会显示 URL：
```
✓ Web UI: https://[your-workspace]--svd-video-ui.modal.run
```

点击链接打开 ComfyUI 界面。

### 步骤 3: 生成第一个视频 (2 分钟)

#### 3.1 准备输入图片
- 下载或准备一张图片（推荐 1024x576 或 576x1024）
- 确保图片清晰、主体明确

#### 3.2 在 ComfyUI 中操作

1. **上传图片**
   - 找到 "LoadImage" 节点
   - 点击 "Choose File" 上传图片

2. **选择模型**
   - 找到 "CheckpointLoaderSimple" 节点
   - 选择 `svd_xt.safetensors` (25 帧高质量版)

3. **调整参数** (可选)
   - Motion Bucket ID: 127 (默认)
   - Steps: 20 (默认)
   - CFG: 2.5 (默认)

4. **生成视频**
   - 点击右侧 "Queue Prompt" 按钮
   - 等待 1-2 分钟生成完成

5. **下载视频**
   - 生成完成后，在输出目录找到 MP4 文件
   - 右键下载视频

## 示例场景

### 场景 1: 人物肖像动画

**输入**: 一张人物肖像照片  
**参数**:
- Motion Bucket ID: 100
- Steps: 25
- Video Frames: 25

**效果**: 人物微表情、眨眼、轻微头部运动

## 常用参数速查

| 场景 | Motion Bucket ID | Steps | 帧数 |
|------|-----------------|-------|------|
| 静态场景 | 30-50 | 20 | 14 |
| 人物肖像 | 80-120 | 25 | 25 |
| 产品展示 | 120-180 | 20 | 14 |
| 动态场景 | 150-200 | 25 | 25 |
| 快速预览 | 127 | 15 | 14 |

## 故障排除

### 问题 1: 视频抖动严重
**解决**: 降低 Motion Bucket ID 到 50-80

### 问题 2: 显存不足
**解决**: 使用 `svd.safetensors` (14 帧版本)

### 问题 3: 生成速度慢
**解决**: 减少 Steps 到 15-20

### 问题 4: 视频质量差
**解决**: 
- 增加 Steps 到 25-30
- 使用 `svd_xt.safetensors`
- 确保输入图片清晰
