## 技术规格

### 模型配置

| 模型 | 文件名 | 大小 | 帧数 | 时长 |
|------|--------|------|------|------|
| SVD 标准版 | svd.safetensors | ~9.5GB | 14 | ~2.3s |
| SVD-XT 扩展版 | svd_xt.safetensors | ~9.5GB | 25 | ~4.2s |
| 图像解码器 | svd_image_decoder.safetensors | ~335MB | - | - |

| 参数 | 值 | 说明 |
|------|-----|------|
| GPU | A100 | 40GB 或 80GB |
| 最大容器数 | 1 | 单实例运行 |
| 并发数 | 5 | 最多 5 个请求 |
| 超时时间 | 86400s | 24 小时 |
| 空闲超时 | 600s | 10 分钟 |
| 端口 | 8000 | Web 界面 |

| Motion Bucket ID | 127 | 1-255 |
| Steps | 20 | 10-50 |

| FPS | 6 | 固定 |
| 分辨率 | 1024x576 | 多种 |

## 部署流程

### 步骤 1: 准备环境
```bash
# 确保已安装 Modal CLI
pip install modal

# 配置 Modal Token
modal token set
```

### 步骤 2: 配置 Secret (可选)
```bash
# 配置 HuggingFace Token
modal secret create huggingface-secret HF_TOKEN=hf_xxx
```

### 步骤 3: 部署服务
```bash
cd "data/projects/SVD 1.1"
modal deploy svd_deploy.py
```

### 步骤 4: 访问服务
```
https://[workspace]--svd-video-ui.modal.run
```
